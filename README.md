# ser (working name)

A portable ground-truth layer for coding agents. Clips onto the agent you already use
(goose, opencode, Claude Code, codex) via its existing hooks; turns your request into
checkable "done" conditions; refuses to let the agent say "done" until reality agrees;
remembers your corrections forever.

See [DESIGN.md](./DESIGN.md).

Status: design. Not a harness, not a UI, not a company — a layer that just works.
